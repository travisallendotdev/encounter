DiceFight

A combat encounter tracker for D&D 5e Dungeon Masters — from campaign setup straight through to initiative-ordered live combat. Below are all six views in the DM's workflow.

Arcane Slate Desktop-first · responsive Static prototype
VIEW 01

Take a Seat

One-field login. The DM types a name — new names conjure a new account. Gamey, not corporate.

Open VIEW 02

Campaigns

The DM's home. Every world as a card, plus a prominent tile to forge a new campaign.

Open VIEW 03

Campaign Detail

Player characters and encounters side by side. Add PCs, spin up encounters, jump into Draft or Active fights.

Open VIEW 04

Encounter Setup

Stage monsters (name, quantity, init modifier) and toggle which heroes are present before combat begins.

Open VIEW 05

Roll for Initiative

Key in each player's roll; let the app roll the monsters or set them by hand. Then begin combat.

Open VIEW 06 · the table screen

Active Encounter

The live tracker. Full turn order high-to-low, players and monsters color-coded, the current turn blown up large and a single Next Turn button.

Walk the flow: Login → Campaigns → Detail → Setup → Initiative → Combat. Each view is its own file.
