## Codebase Patterns
- pnpm requires `CI=true pnpm install` in this environment (no TTY); first run after fresh clone needs this
- better-sqlite3 native addon must be built: run `npm install --ignore-scripts=false` in its pnpm module directory, or set `build-scripts[]=better-sqlite3` in `.npmrc` and rebuild
- Use `import.meta.url` + `fileURLToPath`/`dirname` for `__dirname` equivalent in ESM files
- All local imports need `.js` extension (NodeNext module resolution with verbatimModuleSyntax)
- `db.pragma('foreign_keys = ON')` must be set per-connection for FK enforcement in SQLite
- Hono context Variables (e.g., `dm`) defined in `src/types.ts`; use `new Hono<{ Variables: Variables }>()` for typed context access
- Use `createMiddleware` from `hono/factory` for typed middleware; apply globally in index.ts with path/method exclusion for public routes
- Auth middleware excludes POST /api/auth/login via path check in `app.use('*', ...)` wrapper
